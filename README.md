# 📈 Polynomial Regression Project

## 📌 Overview

This project demonstrates **Polynomial Regression**, a machine learning technique used to model **non-linear relationships** between variables.

Unlike linear regression, polynomial regression fits a curve to the data, making it more suitable for complex datasets.

---

## 🎯 Objective

* Understand non-linear data patterns
* Implement polynomial regression using Python
* Visualize how curve fitting improves prediction accuracy

---

## 🧠 Concepts Covered

* Linear vs Polynomial Regression
* Feature transformation
* Model training and prediction
* Data visualization

---

## 🛠️ Tech Stack

* Python
* NumPy
* Matplotlib
* Scikit-learn

---

## 📊 Project Workflow

1. Import dataset
2. Apply polynomial feature transformation
3. Train regression model
4. Visualize results
5. Compare linear vs polynomial performance
